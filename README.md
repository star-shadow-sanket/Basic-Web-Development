# 🌐 Simple Webpage Project
### Web Development Internship – Task 1 | Apex Planet

## 📌 Project Overview

This project was developed as **Task 1 during my Web Development Internship at Apex Planet**.

The objective of this task was to learn and apply the **fundamental concepts of front-end web development**, including:

- HTML structure
- CSS styling
- Basic JavaScript interactivity

This project demonstrates how a simple webpage can be built using core web technologies while maintaining a clean structure and interactive functionality.

---

## 🎯 Objective

The main objectives of this project were:

- Learn the **basic structure of HTML webpages**
- Apply **CSS styling techniques**
- Implement **basic JavaScript interactivity**
- Understand how **HTML, CSS, and JavaScript work together**

### 2️⃣ CSS3
